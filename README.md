# v0.7.0 版本发布 | 智能分类 + 体验升级

> 首次发布：2026-05-10

## 🎉 版本亮点
本次版本围绕**智能识别**和**体验优化**两大方向，新增了主播身份自动识别、创作者分组移动等重磅功能，同时大幅提升了同步速度和使用流畅度。

## ✨ 新增功能
1. **主播身份智能识别**
   - 自动识别主播,视频创作者,咕咕创作者
   - 智能区分主播型/视频型创作者，按类型显示对应时间信息
   - 停播期主播自动显示多维度时间（投稿+直播+动态）
   - 断更期创作者自动显示最近更新时间

2. **创作者分组移动功能**
   - 鼠标悬停创作者卡片时右上角显示「移动分组」按钮
   - 点击弹出分组选择弹窗，一键移动创作者到目标分组

3. **多维度时间智能显示**
   - 近7天发布的视频时间显示为B站绿色高亮
   - 超过1年未更新的创作者时间自动弱化显示

4. **多种同步模式**
   - 手动触发，拉取所有关注UP主的全部数据
   - 只更新关注分组信息，不触发作品同步

## ⚡ 体验优化
1. **同步速度大幅提升**
   - 同步过程支持后台运行，关闭面板不影响同步进度

2. **使用流程优化**
   - 每日首次打开自动同步，无需手动操作
   - 发布时间固定右对齐，布局更整齐易读

---

## 安装使用
1. 安装油猴脚本管理器（Tampermonkey / Violentmonkey）
2. 下载并安装 `gugu-creator.user.js`
3. 打开任意B站页面，左侧蓝毛女生悬浮图标即可打开面板
4. 快捷键：`Ctrl/Command+Shift+B`

## 注意事项
- 本工具仅用于个人学习和使用，请勿用于商业用途
- 所有数据仅存储在本地浏览器，不上传到任何服务器
- 大量关注首次同步可能需要较长时间
